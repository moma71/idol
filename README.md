## 🧐 작품 소개 (Synopsis)

**"D-30일. 그때까지 개선 못 하면 끝이다."**

JYP 연습생 기숙사 3층. 월말 평가 하위 20%를 기록한 MIRA에게 떨어진 최후 통첩.
막막한 현실 앞에서 우연히 발견한 유튜브 영상 하나가 그녀의 운명을 바꾼다.

**"ChatGPT? 그게 뭔데?"**

절박함 끝에 AI와 손잡은 연습생의 무모하고도 아름다운 도전이 시작된다.

### ✨ 이 소설의 특징

1.  **K-POP 리얼리즘:** 연습생들의 치열한 경쟁과 일상을 담았습니다.
2.  **실전 AI 튜토리얼:** 주인공이 위기를 극복하는 과정에서 사용하는 **'실제 ChatGPT 프롬프트'**를 제공합니다.
3.  **성장 드라마:** 단순한 기술 활용을 넘어, AI를 통해 자신을 마주하고 성장하는 이야기입니다.
