## 🧐 작품 소개 (Synopsis)

**"D-30일. 그때까지 개선 못 하면 끝이다."**

JYP 연습생 기숙사 3층. 월말 평가 하위 20%를 기록한 MIRA에게 떨어진 최후 통첩. 막
막한 현실 앞에서 우연히 발견한 유튜브 영상 하나가 그녀의 운명을 바꾼다.

**"ChatGPT? 그게 뭔데?"**

절박함 끝에 AI와 손잡은 연습생의 무모하고도 아름다운 도전이 시작된다.

### ✨ 핵심 메시지

1. **AI는 도구**: 시스템을 만들어주지만 실행은 사람
2. **꾸준함**: 매일 0.1점씩 100일 = 10점 상승
3. **팀워크**: 혼자는 안 되지만 함께는 가능
4. **투명성**: 데이터 기반 의사결정 (숫자로 말한다)
5. **회복**: 번아웃 방지가 지속 가능성의 핵심

<br>

## 📊 시리즈 통계

- **총 에피소드**: 20개 (완결)
- **총 분량**: ~85,000자
- **기간**: 3개월 (12주)의 기록
- **성장**: 73점(탈락 위기) → 83점(데뷔 확정)

## 🎓 AI Tutorial & Patterns

이 작품은 실제 업무와 학습에 적용 가능한 AI 튜토리얼을 포함하고 있습니다.

### Tool별 활용

| Tool        | Episodes      | 주제                             |
| ----------- | ------------- | -------------------------------- |
| **ChatGPT** | 1, 3, 6       | 첫 대화, 개인 플랜, 팀 밸런스    |
| **Claude**  | 4, 11, 14, 16 | 팀 루틴, 통합 스케줄, 회고, 유지 |
| **Gemini**  | 9             | 멀티모달 분석                    |
| **Sora**    | 7             | 영상 시뮬레이션                  |

### Tutorial Pattern

- **Pattern A (직접 실행)**: 프롬프트 복사/붙여넣기 실습
- **Pattern B (문제 분석)**: AI와 함께 문제의 원인 찾기
- **Pattern C (템플릿)**: 업무/학습용 템플릿 생성
- **Pattern D (체크리스트)**: 실행 여부 점검 및 피드백
