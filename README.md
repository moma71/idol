## 🧐 작품 소개 (Synopsis)

**"새벽 2시 곤지암 허브, AI가 휠소터를 멈췄다."**

BJ대한물류 입사 8년 차, 현장 운영팀 대리 **최도진**. 그의 철학은 '85% 법칙'이다.
완벽한 시스템은 없으니 적당히 돌리고 나머지는 사람의 몸으로 때운다는 것.

하지만 어느 날, 본사에서 날아온 메일 한 통이 그의 새벽을 뒤흔든다. **[AI
물류혁신 TF 발령]**

현장을 모르는 본사 엘리트 동기 **정민수**와, 이론만 완벽한 **AI 시스템**. 그 사
이에서 도진은 살아남기 위해 '생존형 프롬프트'를 깎기 시작하는데...

### ✨ 이 소설의 특징

1.  **하이퍼 리얼리즘:** 물류센터의 새벽 공기와 현장의 치열함을 담았습니다.
2.  **실전 AI 튜토리얼:** 각 에피소드마다 주인공이 위기를 해결한 **'실제
    ChatGPT/Claude 프롬프트'**를 제공합니다.
3.  **인터랙티브 웹:** 단순 텍스트가 아닌 영상, 사운드, 반응형 요소가 결합된 웹
    경험을 제공합니다.
