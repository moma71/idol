## 🎯 Overall Arc Structure (4 Phases)

### Phase 1: 위기와 발견 (Episode 1-5)
**Theme**: "데뷔 못 하면 끝이야"
**Tension**: 탈락 위기, 실력 부족, 절망
**AI Discovery**: 우연히 발견 → 처음 사용 → 효과 확인
**Character State**: 
- MIRA: 탈락 통보 직전
- ARIN: 독불장군
- CHAEWON: 과도한 부담
- LUNA: 비주얼만 평가받음
- REI: 정체성 혼란

**SIGNATURE PATTERNS 적용:**
- Pattern 1: 극단적 생존 설정 (탈락 = 꿈의 끝)
- Pattern 2: 리얼한 연습생 디테일
- Pattern 4: 타이머/카운트다운 (D-30일까지 데뷔조 발표)

---

### Phase 2: 성장과 갈등 (Episode 6-10)
**Theme**: "혼자서는 안 돼"
**Tension**: 개인 성장 vs 팀워크, AI 의존 vs 진짜 실력
**AI Integration**: 적극 활용 → 의존 우려 → 균형 찾기
**Character State**:
- MIRA: 실력 향상 시작
- ARIN: 팀워크 실패 경험
- CHAEWON: 번아웃 위기
- LUNA: 실력 증명 시도
- REI: SNS 비교 지옥

**SIGNATURE PATTERNS 적용:**
- Pattern 3: 웃프게 (AI 덕분에 살았다는 자조)
- Pattern 5: 집단 vs 개인 (팀 점수 vs 개인 평가)

---

### Phase 3: 변화와 깨달음 (Episode 11-15)
**Theme**: "우리가 진짜야"
**Tension**: 최종 평가 임박, 데뷔조 경쟁, 진짜 실력 vs AI
**AI Balance**: AI는 도구 → 진짜 실력 구축 → 자립
**Character State**:
- MIRA: 자신감 회복
- ARIN: 협력 깨달음
- CHAEWON: 여유 찾기
- LUNA: 실력 향상
- REI: 자기 색깔 찾기

**SIGNATURE PATTERNS 적용:**
- Pattern 2: 리얼한 디테일 (최종 평가 과정)
- Pattern 4: 타이머 (D-7일, D-3일)

---

### Phase 4: 완성과 데뷔 (Episode 16-20)
**Theme**: "우리는 해냈어"
**Tension**: 최종 평가, 데뷔조 발표, 진짜 시작
**AI Role**: 보조 도구 → 이제 진짜 실력이 메인
**Character State**:
- MIRA: 진짜 실력 확립
- ARIN: 진정한 에이스
- CHAEWON: 성숙한 리더
- LUNA: 올라운더 인정
- REI: 정체성 확립

**SIGNATURE PATTERNS 적용:**
- Pattern 3: 웃프게 (데뷔했지만 이제 시작)
- Pattern 5: 개인 성장이 팀 성공으로

## 📅 Episode-by-Episode Breakdown

#### Episode 1: "D-30일, 탈락 통보"
**타이틀**: 첫 월말 평가의 악몽

**Main Plot:**
- 월말 평가 실시
- MIRA 최하위 (탈락 경고)
- "다음 평가까지 개선 안 되면 퇴소"
- 절망에 빠진 MIRA

**AI Tutorial**: ChatGPT로 자기 강약점 분석
- Problem: 뭘 연습해야 할지 모름
- Solution: ChatGPT에 평가 피드백 입력 → 분석 → 개선 계획
- Tool: ChatGPT

**Character Focus:**
- MIRA: 주인공, 탈락 위기
- CHAEWON: 1등이지만 부담감 큼
- ARIN: 2등, 여유로움

**Key Scene:**
- Opening: 평가 결과 발표 (긴장감)
- Climax: MIRA 탈락 경고 ("다음 평가가 마지막")
- Tutorial: 혼자 남은 MIRA, 우연히 ChatGPT 발견
- Ending: "이거라도 해보자" 결심

**Learning Concept**: Level 1 - AI 협업의 시작

**Emotional Arc**: 절망 → 희미한 희망

**Illustration Count**: 6개 (Standard)

**SIGNATURE PATTERNS:**
- Pattern 1: 극단적 생존 (탈락 = 끝)
- Pattern 4: 타이머 (D-30일)

---

#### Episode 2: "새벽 5시의 비밀"
**타이틀**: MIRA의 첫 성공

**Main Plot:**
- MIRA, ChatGPT 분석 따라 연습

- 작은 성공 (보컬 점수 상승)

**AI Tutorial**: Claude로 보컬 연습 계획 수립
- Problem: 효율적 연습 방법 모름
- Solution: Claude에 목표 입력 → 3주 연습 계획
- Tool: Claude

**Character Focus:**
- MIRA: 변화 시작
- ARIN: MIRA 변화 눈치챔 (궁금)
- LUNA: MIRA 응원

**Key Scene:**
- Opening: MIRA 새벽 연습 (땀 범벅)
- Climax: 트레이너 칭찬 ("많이 좋아졌어")
- Tutorial: 더 체계적으로 하고 싶음 → Claude 발견
- Ending: ARIN "뭐 하는 거야?" (호기심)

**Learning Concept**: Level 1 - 체계적 학습 계획

**Emotional Arc**: 희망 → 작은 성공

**Illustration Count**: 7개 (Standard)

**SIGNATURE PATTERNS:**
- Pattern 2: 리얼한 디테일 (새벽 5시 연습)
- Pattern 3: 웃프게 (AI 덕분이라는 자조)

#### Episode 3: "첫 작사 미션"
**타이틀**: 팀 프로젝트의 위기

**Main Plot:**
- 자작곡 미션 발표 (3일 기한)
- 5명 아무도 작사 못 함
- CHAEWON 부담감 폭발 "내가 다 해야 해"
- MIRA "Claude 써보는 거 어때?"

**AI Tutorial**: Claude로 K-POP 작사
- Problem: 3일 안에 가사 완성 필요
- Solution: Claude에 콘셉트 입력 → 초안 생성 → 팀 수정
- Tool: Claude

**Character Focus:**
- 전원 등장 (팀 프로젝트)
- CHAEWON: 혼자 떠안으려 함
- ARIN: "난 작사 못 해" 방관
- MIRA: AI 제안

**Key Scene:**
- Opening: 미션 발표 (3일, 압박감)
- Climax: CHAEWON 번아웃 직전
- Tutorial: MIRA가 Claude로 초안 생성
- Ending: "오... 이거 쓸만한데?" 팀 협력 시작

**Learning Concept**: Level 1 - AI 창작 협업

**Emotional Arc**: 압박 → 협력 시작

**Illustration Count**: 8개 (Rich - 팀 협력 장면 많음)

**SIGNATURE PATTERNS:**
- Pattern 2: 작사 디테일 (라임, 구조)
- Pattern 5: 집단 vs 개인 (팀 미션이지만 각자 기여)

---

#### Episode 4: "비주얼만 예쁘다고?"

**Main Plot:**
- 중간 평가 (팀 발표)
- LUNA 비주얼 칭찬, 실력 비판

**AI Tutorial**: ChatGPT로 보컬 트레이닝 플랜
- Problem: 보컬 약점 개선 방법 모름
- Solution: ChatGPT에 약점 입력 → 맞춤 트레이닝
- Tool: ChatGPT

**Character Focus:**
- LUNA: 메인, 비주얼 콤플렉스
- MIRA: 위로하며 ChatGPT 알려줌
- ARIN: 무심한 듯 걱정

**Key Scene:**
- Opening: 팀 발표 (성공적)
- Climax: 평가 "LUNA는 얼굴만..." 충격
- Tutorial: MIRA가 LUNA에게 ChatGPT 소개
- Ending: LUNA "나도 할 수 있어" 결심

**Learning Concept**: Level 1 - 약점 분석과 개선

**Emotional Arc**: 좌절 → 결심

**Illustration Count**: 7개 (Standard)

**SIGNATURE PATTERNS:**
- Pattern 3: 웃프게 (예쁘다는 게 독인 상황)
- Pattern 2: 평가 디테일 (점수, 코멘트)

#### Episode 5: "D-20일, 두 번째 평가"
**타이틀**: 작은 기적들

**Main Plot:**
- 두 번째 월말 평가

**Character Focus:**

**Key Scene:**

- Climax: MIRA, LUNA 점수 상승
- Tutorial: REI 혼자 작곡하며 Claude 사용
- Ending: "D-20일... 우리 해볼 만한데?" 희망

**Learning Concept**: Level 2 - AI 도구 확산

**Emotional Arc**: 불안 → 희망

**Illustration Count**: 6개 (Standard)

**SIGNATURE PATTERNS:**
- Pattern 4: 타이머 (D-20일)
