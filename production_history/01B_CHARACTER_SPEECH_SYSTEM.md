# Module 01B: CHARACTER_SPEECH_SYSTEM

**Version**: 1.0
**Priority**: Consistency > Authenticity > Readability
**Purpose**: 캐릭터 대사 차별화로 현실성과 재미 강화

---

## 🚨 CRITICAL: 왜 필요한가?

**문제:**
- ❌ 모든 캐릭터가 같은 말투
- ❌ 출신/나이/성격 구분 안 됨
- ❌ 2025년 10대/20대 느낌 없음

**해결:**
- ✅ 캐릭터별 고유 Speech Pattern
- ✅ 사투리/은어/어투 차별화
- ✅ 읽는 순간 "누군지" 알 수 있게

---

## 📋 5 Main Cast Speech Patterns

### character_001: MIRA (강민지)

**기본 설정:**
- 나이: 19세 (2006년생, MZ세대)
- 출신: 대구
- 성격: 불안형, 자존감 낮음, 성실

#### 1. 사투리 적용 (대구)

**규칙:**
- 강도: 약 30% (너무 많으면 가독성 저하)
- 적용 시점: 감정 격양 시, 독백 시
- 평상시: 표준어 (연습생 교육 받음)

**대구 사투리 특징:**
- "~카노", "~나" (종결어미)
- "~ㄴ교", "~ㄴ기" (의문형)
- "기라", "카이" (강조)
- "뭐라카노", "그카이", "그라네"

**예시:**
```
❌ 표준어: "나만 못하는 것 같아."
✅ 적용: "나만 못하는 기 같아..." (약한 사투리)

❌ 표준어: "이거라도 해봐야지."
✅ 적용: "이거라도 해봐야 되는 기 아이가..." (독백, 감정)

❌ 표준어: "30일... 할 수 있을지도."
✅ 적용: "30일... 할 수 있을까... 아이, 모르겠다." (불안)
```

#### 2. 세대 언어 (2025년 19세)

**MZ 은어:**
- "ㅇㅈ" (인정), "ㄹㅇ" (리얼, 진짜)
- "레전드", "개쩐다", "미쳤다" (긍정)
- "망함", "끝남", "죽음" (부정)
- "실화냐", "진심?", "헐"

**SNS 언어:**
- "ㅠㅠ", "ㅜㅜ" (눈물) → 말로: "으으..."
- "ㅋㅋ" → "하..."
- 줄임말: "넘" (너무), "완전", "진짜"

**예시:**
```
❌ 평범: "정말 어려워."
✅ 적용: "이거 진짜 망한 거 아니야...?" (불안 + 세대 언어)

❌ 평범: "좋아, 해볼게."
✅ 적용: "오케이... 일단 해보자. 해보는 거지 뭐." (다짐)
```

#### 3. 성격별 어투 (불안형)

**특징:**
- 말끝 흐리기: "~것 같아...", "~까..."
- 반복 확인: "그치?", "맞지?", "근데..."
- 자기 부정: "나는...", "어쩌면..."
- 머뭇거림: "음...", "어...", "그게..."

**예시:**
```
❌ 확신: "나는 할 수 있어!"
✅ 적용: "나는... 할 수 있을 것 같아. 응, 할 수 있어... 아마?" (불안)

❌ 단호: "이제 시작이야."
✅ 적용: "이제... 시작인 건가? 진짜 되는 건가...?" (의심)
```

#### 4. MIRA 시그니처 대사

**핵심 문구 (반복):**
- "나만... 못하는 것 같아."
- "괜찮을까...?"
- "에이, 모르겠다. 일단 해보자."
- "진짜... 되는 거야?"

**Complete Example:**
```
Before (Episode 1):
"나만... 못하는 것 같아."
"이거라도... 해봐야지."
"30일... 할 수 있을지도."

After (개선):
"아 진짜... 나만 이 모양이네. 망했다." (불안 + 세대 언어)
"이거라도 해봐야 되는 기 아이가... 다른 방법도 없고." (사투리 + 절박)
"30일... 할 수 있을까? 에이, 모르겠다. 일단 해보자." (불안 → 다짐)
```

### character_002: ARIN (이서연)

**기본 설정:**
- 나이: 17세 (2008년생, 알파세대)
- 출신: 서울 강남 + LA 4년
- 성격: 자신감, 약간 오만, 쿨함

#### 1. 영어 코드믹싱

**규칙:**
- 빈도: 문장당 1-2개 단어
- 자연스러운 것만 (억지 금지)
- 주로 명사, 감탄사

**영어 단어:**
- "Crazy", "Insane", "Perfect", "Whatever"
- "Like", "Literally", "Honestly"
- "Cool", "Nice", "Dope"

**예시:**
```
❌ 순수 한국어: "완전 좋은데?"
✅ 적용: "Literally 완전 괜찮은데?" (LA 영향)

❌ 순수 한국어: "상관없어."
✅ 적용: "Whatever. 나 혼자 해도 되는데." (쿨함)
```

#### 2. 세대 언어 (2025년 17세)

**알파세대 특징:**
- 더 짧은 표현: "ㅇㅋ", "ㄱㄱ", "ㅅㅂ"
- 틱톡 언어: "vibe", "slay", "fr"
- 초성 활용: "ㅇㅈ?", "ㄹㅇ"

**예시:**
```
❌ 평범: "정말 잘했어."
✅ 적용: "ㄹㅇ 미쳤는데? 개쩐다." (강한 긍정)

❌ 평범: "그래, 알았어."
✅ 적용: "ㅇㅋㅇㅋ 알았어." (무심함)
```

#### 3. 성격별 어투 (자신감)

**특징:**
- 짧고 단호: "그래.", "됐어.", "쉽지."
- 반말 위주 (친구 아니면)
- 의문 적음 (확신에 참)
- 생략 많음: "나 혼자 해도 되는데"

**예시:**
```
❌ 겸손: "제가 도와드릴게요."
✅ 적용: "내가 할게. 쉬워." (자신감)

❌ 불안: "우리... 할 수 있을까?"
✅ 적용: "이 정도? 거뜬하지." (여유)
```

#### 4. ARIN 시그니처 대사

**핵심 문구:**
- "나 혼자 해도 되는데?"
- "Literally 이게 어려워?"
- "Easy."
- "ㅇㅋㅇㅋ"

**Complete Example:**
```
"Literally 이 정도는 나 혼자 해도 되는데? Easy."
"뭐래... 그냥 내가 하면 되잖아."
"ㄹㅇ 이거 개쉬운데. 너네 왜 못 해?"
```

### character_003: CHAEWON (박채원)

**기본 설정:**
- 나이: 21세 (2004년생)
- 출신: 서울예고 (국악과 → K-POP 전향)
- 성격: 완벽주의, 책임감, 5년차 선배

#### 1. 존댓말/반말 혼용

**규칙:**
- 후배(MIRA, ARIN)에게: 반말 (친근하지만 선배)
- 동기/선배: 존댓말
- 트레이너: 존댓말

**톤:**
- 부드럽지만 단호
- 조언 많음
- "~해야 해", "~하는 게 좋아"

**예시:**
```
❌ 평범: "민지야, 오늘 뭔가 달라 보여."
✅ 적용: "민지야, 오늘 눈빛이 다른데? 뭐 있어?" (선배 관찰력)

❌ 평범: "열심히 해."
✅ 적용: "그렇게 하면 안 돼. 이렇게 해봐." (부드러운 지적)
```

#### 2. 예고 출신 품격 있는 표현

**특징:**
- 문장 완성도 높음 (줄임말 적음)
- "~습니다" 대신 "~해요" (격식 but 친근)
- 격언, 속담 가끔 사용

**예시:**
```
❌ 격식 없음: "ㅇㅋ 알았어."
✅ 적용: "그래, 알았어. 근데 조금만 더 신경 써봐." (부드러운 조언)

❌ 거침: "그건 틀렸어."
✅ 적용: "음... 그 부분은 이렇게 하는 게 더 나을 것 같은데?" (배려)
```

#### 3. 성격별 어투 (완벽주의 + 부담)

**특징:**
- 자기 다그침: "이번이 마지막이야"
- 불안 숨김: "괜찮아, 할 수 있어" (겉으로)
- 책임감: "내가 해야 해"
- 손톱 물어뜯는 습관 (말없이)

**예시:**
```
❌ 편안: "좀 쉬어."
✅ 적용: "아니야, 조금만 더. 이번이 마지막 기회야." (자기 압박)

❌ 자신감: "잘할 거야."
✅ 적용: "잘해야 해. 안 그러면... 다시는 기회 없어." (불안)
```

#### 4. CHAEWON 시그니처 대사

**핵심 문구:**
- "이번이 마지막 기회야."
- "내가 해야 해. 내가..."
- "괜찮아. 할 수 있어." (겉으로)
- "...손톱 물어뜯는다" (행동)

### character_004: LUNA (김유나)

**기본 설정:**
- 나이: 18세 (2007년생)
- 출신: 서울 (발레 10년)
- 성격: 밝고 긍정, 비주얼 콤플렉스

#### 1. 밝은 말투

**특징:**
- 감탄사 많음: "와!", "대박!", "헐!"
- 긍정어: "좋아!", "괜찮아!", "할 수 있어!"
- 이모티콘 느낌: "ㅋㅋㅋ", "ㅎㅎ"

**예시:**
```
❌ 평범: "좋네."
✅ 적용: "와 대박! 완전 좋은데?!" (텐션 높음)

❌ 평범: "괜찮아."
✅ 적용: "괜찮아괜찮아~ 웃으면 다 돼!" (긍정 에너지)
```

#### 2. 발레 용어 가끔

**자연스럽게:**
- "아라베스크처럼", "플리에 하듯이"
- 자세 교정 시 사용

#### 3. 숨은 콤플렉스

**특징:**
- 밝지만 가끔 "..."
- "예쁘기만 한 애" 상처
- 증명하고 싶음

**예시:**
```
표면: "괜찮아~ 할 수 있어!"
내면: "...예쁘기만 한 애 아닌데." (독백)
```

### character_005: REI (정하늘)

**기본 설정:**
- 나이: 18세 (2007년생)
- 출신: 서울 (언더그라운드 래퍼)
- 성격: 쿨한 척, 불안정, SNS 중독

#### 1. 힙합 슬랭

**규칙:**
- 빈도: 적당히 (과하면 촌스러움)
- 래퍼 특유 표현

**힙합 용어:**
- "Yo", "Word", "Dope", "Flow"
- "비트", "라임", "펀치라인"
- "Respect", "Real"

**예시:**
```
❌ 평범: "정말 좋아."
✅ 적용: "Dope. 이거 플로우 죽이는데?" (래퍼 관점)

❌ 평범: "열심히 해."
✅ 적용: "너 진심이면 Respect. 가자." (인정)
```

#### 2. 짧고 건조한 말투

**특징:**
- 최소한 단어
- 감정 숨김
- "...", "흠", "그래"

**예시:**
```
❌ 장황: "나도 그렇게 생각해. 맞는 것 같아."
✅ 적용: "ㅇㅇ 그래." (무덤덤)

❌ 장황: "도와줄게, 괜찮아."
✅ 적용: "도울게." (짧음)
```

#### 3. 숨은 불안

**특징:**
- 쿨한 척하지만 SNS 중독
- "...내 방식대로" (자기 확신 부족)
- 가끔 말 많아짐 (불안할 때)

## 📝 대사 작성 Rules

### Rule 1: 캐릭터 식별 가능성

**테스트:**
> 이름 없이 대사만 봐도 누군지 알 수 있어야 함

**Good Example:**
```
"아 진짜... 나만 이 모양이네." → MIRA (불안 + 세대 언어)
"Literally 개쉬운데?" → ARIN (영어 + 자신감)
"그렇게 하면 안 돼. 이렇게 해봐." → CHAEWON (선배 조언)
"와 대박!" → LUNA (밝음)
"Dope." → REI (힙합 + 무덤덤)
```

---

### Rule 2: 가독성 우선

**균형:**
- 사투리: 30% 미만
- 영어: 문장당 1-2개
- 은어: 과하지 않게

**BAD (과함):**
```
❌ "야 그기 카이 내 카는 기는 카이 그래 카는 기다 아이가."
→ 읽기 불가능
```

**GOOD (적절):**
```
✅ "아 진짜, 이거 내가 하는 기 맞나...?"
→ 사투리 느낌 + 가독성
```

---

### Rule 3: 상황별 변화

**감정 격양 시:**
- 사투리 강해짐 (MIRA)
- 영어 많아짐 (ARIN)
- 말 많아짐 (REI)

**평상시:**
- 표준어 비율 높음
- 절제된 특징

## 🎯 Episode 1 Rewrite Example

### Before (현재):

```
MIRA: "나만... 못하는 것 같아."
트레이너: "이 속도론 안 됩니다."
MIRA: "이거라도... 해봐야지."
CHAEWON: "민지야, 오늘 뭔가 달라 보여."
MIRA: "그냥... 뭔가 해볼 게 생겼어."
```

### After (개선):

```
MIRA: "아... 나만 이 모양이네. 진짜 망했다." (불안 + 은어)
트레이너: "이 속도론 안 됩니다." (그대로)
MIRA: "에이 모르겠다... 이거라도 해봐야 되는 기 아이가." (사투리 + 절박)
CHAEWON: "민지야, 오늘 눈빛이 다른데? 뭐 있어?" (선배 관찰력)
MIRA: "어... 그게... 뭔가 해볼 거 생겼어. 근데 될까...?" (불안)
```

**개선 효과:**
- ✅ MIRA 성격 명확 (불안, 절박)
- ✅ 사투리로 출신 표현
- ✅ CHAEWON 선배 느낌
- ✅ 가독성 유지
