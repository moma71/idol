## 💡 Hands-On Tutorial: MIRA의 ChatGPT 자기분석

**Real-world situation**: MIRA는 월말 평가에서 74점(하위 20%)을 받고 30일 내 개선 못 하면 퇴소 통보를 받았다. 무엇부터 고쳐야 할지 막막한 상황. ChatGPT로 자신의 강약점을 객관적으로 분석해서 효율적인 학습 계획을 세우기로 했다.

**Copy-paste prompt**:
```
당신은 K-POP 연습생 트레이닝 전문가입니다.
다음 연습생의 상황을 분석해주세요:

이름: {이름, 예: "민지"}
나이: {나이, 예: "19세"}
연습생 기간: {기간, 예: "3년"}
포지션: {포지션, 예: "서브보컬"}

최근 평가 결과:
- 보컬: {점수, 예: "72점"}
- 댄스: {점수, 예: "76점"}
- 랩: {점수, 예: "65점"}
- 종합: {점수, 예: "74점"} (하위 {순위, 예: "20%"})

현재 문제:
{문제 상황, 예: "댄스 동작이 깔끔하지 않음, 박자는 맞는데 어색함"}

시간: {남은 기간, 예: "30일"} 안에 개선 필요

다음을 분석해주세요:
1. 3가지 핵심 약점 (우선순위 순서로)
2. 각 약점의 구체적 원인
3. 30일 안에 개선 가능한 실행 계획
4. 매주 체크할 마일스톤

출력 형식: Markdown 표
```

**What you'll get**: 3가지 핵심 약점과 각각의 원인 분석, 30일 개선 계획을 표로 정리한 결과. 우선순위가 명확하고 실행 가능한 구체적 액션 아이템이 포함되어 있어 바로 실천할 수 있습니다.

**Try it yourself**:
- [ ] https://chat.openai.com 접속 (새 탭에서 열기)
- [ ] "Sign up" 버튼 클릭 → Google 계정으로 가입 (무료)
- [ ] 가입 완료 후 화면 하단의 채팅 입력창 확인
- [ ] 위의 프롬프트 전체를 복사 (Ctrl+C 또는 드래그 후 우클릭)
- [ ] 채팅창에 붙여넣기 (Ctrl+V)
- [ ] `{이름, 예: "민지"}` → 자신의 이름으로 수정 (예: "수진")
- [ ] `{나이, 예: "19세"}` → 자신의 나이로 수정
- [ ] `{점수, 예: "72점"}` → 자신의 실제 평가 점수로 수정 (각 항목별로)
- [ ] `{문제 상황, 예: "..."}` → 자신이 느끼는 구체적인 문제 상황 작성
- [ ] Enter 키를 눌러 실행
- [ ] 결과가 나오면 전체 선택 (Ctrl+A) → 복사 (Ctrl+C)
- [ ] 노트 앱(메모장, Notion 등)에 붙여넣어 저장
- [ ] 표를 보며 오늘부터 실천할 액션 1개 선택

**Example result**:
```
| 우선순위 | 약점 | 원인 분석 | 개선 방법 | 주차별 목표 |
|---------|------|----------|----------|-------------|
| 1 | 댄스 동작 마무리 부족 | 코어 근력 약함, 동작 끝까지 힘 안 들어감 | 매일 플랭크 3분, 동작마다 마지막 1초 멈춤 연습 | 1주: 플랭크 1분 → 4주: 3분 |

**주차별 체크포인트:**
- 1주차: 기초 체력 향상 (플랭크 1분 달성)
```

## 🎯 Learning Concept: "나는 할 수 있어"

**Level 1 Concept**: AI 협업의 시작

MIRA는 탈락 위기에서 ChatGPT를 발견했다. 막연한 불안을 구체적인 실행 계획으로 바꿨다. "나만 못하는 것 같아"에서 "나는 할 수 있어"로 바뀐 순간이다.

AI는 마법이 아니다. 답을 주는 게 아니라 질문을 정리해준다. "뭘 해야 할지 모르겠어"를 "이걸 이렇게 하면 돼"로 바꿔준다. MIRA의 절박함이 도구를 만났을 때, 변화가 시작됐다.

**Why it matters in K-POP 연습생 시스템**: 연습생들은 매달 평가를 받지만 정확히 "뭘" 고쳐야 할지 모르는 경우가 많다. "더 잘해"는 알겠는데 "어떻게"가 없다. AI 도구는 막연한 피드백을 구체적 액션으로 만들어준다. 30일이라는 압박 속에서 MIRA에게 가장 필요했던 건 "우선순위"였다. ChatGPT는 그걸 줬다.

**Remember (MIRA's words)**: "이거라도... 해봐야지."
