# 05B_HIT_CONTENT_VALIDATION.md

**Version**: 1.0 
**Priority**: CRITICAL - 매 에피소드 강제 실행
**Purpose**: 흥행 요소 검증 = 생명선

---

## 🚨 CRITICAL: 왜 이 검증이 필요한가?

**문제**:
- ❌ 기술적으로 완벽해도 재미없으면 실패
- ❌ Episode 1만 체크 → 나머지 20화 놓침
- ❌ 일관성 없으면 독자 이탈

**해결**:
- ✅ 매 에피소드마다 강제 검증
- ✅ 점수 85점 미만 = 즉시 재작성
- ✅ SIGNATURE PATTERNS 3개 미만 = FAIL

**핵심**:
> **흥행 요소 = 생명선. 이거 놓치면 끝.**

---

## 📋 Hit Content Validation (120점 척도)

### ⭐ Step 1: SIGNATURE PATTERNS (50점)

**각 Pattern별 0-10점 채점:**

#### Pattern 1: 극단적 생존 설정 (10점)

**질문:**
- [ ] "탈락 = 끝이야" 느낌 전달되는가?
- [ ] 독자가 긴장감 느끼는가?
- [ ] 생존이 걸린 순간이 있는가?

**점수**: [0-10]
- 8-10점 = ✅ 강함 (생존 압박 강렬)
- 5-7점 = ⚠️ 약함 (언급만 됨)
- 0-4점 = ❌ 없음 (생존 요소 없음)

**좋은 예:**
```
"다음 평가에서 개선 안 되면 퇴소"
트레이너의 말이 MIRA의 귀에 울렸다.
퇴소 = 꿈의 끝. 돌아갈 곳은 대구 치킨집뿐.
```

**나쁜 예:**
```
평가 결과가 좋지 않았다.
조금 더 노력해야겠다고 생각했다.
```

---

#### Pattern 2: 리얼한 연습생 디테일 (10점)

**질문:**
- [ ] 평가 점수 구체적인가? (85점, 92점 등)
- [ ] 시간/장소 디테일 있는가? (새벽 5시, 3층 연습실)
- [ ] "진짜 연습생" 느낌 나는가?

**점수**: [0-10]
- 8-10점 = ✅ 강함 (디테일 풍부)
- 5-7점 = ⚠️ 약함 (일부만 구체적)
- 0-4점 = ❌ 없음 (전부 추상적)

**좋은 예:**
```
보컬: 72점 (전월 68점 +4)
댄스: 85점 (전월 83점 +2)
랩: 65점 (전월 64점 +1)
종합: 74점 (하위 20%)
```

**나쁜 예:**
```
평가 점수가 낮았다.
```

---

#### Pattern 3: 웃프게 (감정선) (10점)

**질문:**
- [ ] 울면서 웃는 순간 있는가?
- [ ] "웃프다" 느낌 전달되는가?
- [ ] 감정 대조 (밝음 vs 슬픔) 명확한가?

**점수**: [0-10]
- 8-10점 = ✅ 강함 (웃프게가 강렬)
- 5-7점 = ⚠️ 약함 (약간 느껴짐)
- 0-4점 = ❌ 없음 (감정 단조로움)

**좋은 예:**
```
"ChatGPT한테 물어볼 정도로 절박했구나..."
MIRA는 웃었다. 눈물이 났지만 웃었다.
AI한테 인생 상담하는 연습생이라니.
```

**나쁜 예:**
```
조금 슬펐지만 괜찮았다.
```

---

#### Pattern 4: 타이머/카운트다운 (10점)

**질문:**
- [ ] "D-X일" 명확히 표시되는가?
- [ ] 긴박감 조성되는가?
- [ ] 시간 압박 느껴지는가?

**점수**: [0-10]
- 8-10점 = ✅ 강함 (타이머 명확)
- 5-7점 = ⚠️ 약함 (한 번만 언급)
- 0-4점 = ❌ 없음 (시간 요소 없음)

**좋은 예:**
```
# Episode 1: D-30일, 탈락 통보

"다음 평가까지 30일"
달력을 보던 MIRA의 손이 떨렸다.
30일 후 = 생사 판가름.
```

**나쁜 예:**
```
곧 평가가 있다.
```

---

#### Pattern 5: 집단 vs 개인 갈등 (10점)

**질문:**
- [ ] 개인 vs 팀 긴장 있는가?
- [ ] "혼자 vs 함께" 고민 보이는가?
- [ ] 갈등 해소 과정 있는가?

**점수**: [0-10]
- 8-10점 = ✅ 강함 (갈등 명확)
- 5-7점 = ⚠️ 약함 (언급만)
- 0-4점 = ❌ 없음 (집단 요소 없음)

**좋은 예:**
```
"내가 탈락하면 4명이 데뷔하나..."
MIRA는 룸메이트들을 봤다.
다 같이 못 가는 거라면, 차라리...
```

**나쁜 예:**
```
팀워크가 중요하다.
```

## 🚨 CRITICAL RULES

### Rule 1: 흥행 검증 = 최우선

**검증 순서:**
```
1. Hit Content Validation (120점) ⭐ FIRST
   ↓
   85점 미만? → 즉시 재작성
   ↓
2. Technical Validation (Category 1-7)
   ↓
3. AI Tutorial Integration (Category 9)
```

**이유**:
- 기술 완벽 + 재미없음 = 실패
- 흥행 요소가 핵심
- 기술은 나중에 수정 가능, 재미는 근본부터

---

### Rule 2: 매 에피소드 강제 실행

**적용 대상:**
- ✅ Episode 1, 2, 3... 20 전부
- ✅ 예외 없음
- ✅ 점수 기록 필수

**Skip 금지:**
- ❌ "이번 화는 괜찮을 것 같아" → Skip 금지
- ❌ "시간 없어서" → Skip 금지
- ❌ "대충 봤는데" → 정확히 채점

---

### Rule 3: 85점 미만 = 즉시 재작성

**수정 vs 재작성:**

**수정 가능 (85-99점):**
- 문장 다듬기
- 디테일 추가
- 대사 개선

**재작성 필요 (85점 미만):**
- 근본적 재구성
- Pattern 추가
- 갈등 강화
- 전체 톤 변경

**85점 = 마지노선**

---

### Rule 4: SIGNATURE PATTERNS 3개 미만 = FAIL

**판정:**
```
Pattern 8점 이상 개수:
- 5개: EXCELLENT ⭐⭐⭐⭐⭐
- 4개: GREAT ⭐⭐⭐⭐
- 3개: PASS ✅ (최소 기준)
- 2개: FAIL ❌ (재작성)
- 0-1개: CRITICAL FAIL ❌❌ (전면 재작성)
```

**왜 3개?**
- "데뷔 못 하면 죽는 병 걸림" 성공 요인
- 최소 이 정도는 있어야 독자 몰입
- 5개 전부는 어려워도 3개는 필수

## ⚠️ Emergency Actions

### 만약 Episode N이 70점 미만이면?

**즉시 실행:**

1. **STOP ALL WORK**
   - 다음 에피소드 쓰지 말 것
   - 현재 에피소드 문제 해결 우선

2. **Root Cause 분석**
   - 어떤 Pattern이 약한가?
   - 어떤 차원이 부족한가?
   - 재미 요소가 뭐가 없는가?

3. **전면 재작성**
   - 수정으로 안 됨
   - 처음부터 다시
   - Pattern 3개 이상 강하게

4. **재검증**
   - 120점 척도 다시
   - 85점 이상 확인
   - 그래도 안 되면 또 재작성

## 💡 Best Practices

### Pattern 강하게 적용하는 법

**Pattern 1 (생존):**
- "탈락 = 끝" 명시적 언급
- 독백: "이게 마지막 기회야"
- 타인 시선: "저러다 정말 나가겠네"

**Pattern 2 (디테일):**
- 숫자: 점수, 시간, 날짜
- 장소: 3층 연습실 A룸
- 소품: 땀 범벅 수건, 메모 노트

**Pattern 3 (웃프게):**
- 울면서 웃기
- 밝은 말투 + 슬픈 상황
- "아이러니" 상황 만들기

**Pattern 4 (타이머):**
- 에피소드 제목에 D-X
- 본문에 최소 2번 언급
- 압박감 조성

**Pattern 5 (갈등):**
- 개인 목표 vs 팀 목표
- "나만 살까 vs 같이 갈까"
- 갈등 → 해소 과정
