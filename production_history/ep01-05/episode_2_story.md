# Episode 2: ARIN의 의심

![Opening - 새벽 복도에서 마주친 MIRA와 ARIN](placeholder)
*새벽 6시. 3층 복도. 땀 흘린 MIRA가 연습실에서 나오다가 에어팟 낀 ARIN과 마주친다. 금발의 ARIN이 무표정하게 MIRA를 본다.*

2025년 12월 24일 화요일, 새벽 6시. D-29일.

민지는 새벽 5시부터 2시간 연습을 마치고 복도로 나왔다. ChatGPT 계획표대로였다.

"후... 호흡이 진짜 좋아진 거 같아..."

그때 복도 끝에서 누군가 걸어왔다. 금발 생머리. 에어팟. Nike 크롭티. ARIN이었다.

둘이 마주 봤다.

"...야." ARIN이 먼저 말했다. 에어팟 한쪽을 뺐다.

"어... 응. 안녕..."

민지는 메모 노트를 가슴에 꼭 안았다.

ARIN의 시선이 노트로 향했다.

"너 요즘 뭐 해? Literally 매일 새벽에 보이는데."

"그냥... 연습..."

"연습? 다들 하는데. 근데 너는 뭔가 다른 거 같은데?"

ARIN이 한 걸음 다가왔다. 민지는 뒤로 물러섰다.

"다, 다르지 않아..."

"흠." ARIN은 민지를 위아래로 훑었다.

"그 노트. 뭐 적어? 보여줘."

"아, 아니야! 그냥 개인 거라..."

민지는 노트를 등 뒤로 숨겼다.

ARIN의 눈이 가늘어졌다.

"...수상한데. 너 뭔가 숨기는 거 있지?"

"없어! 진짜 없어!"

"Whatever. 근데 말이야..." ARIN이 에어팟을 다시 꼈다.

"요즘 네 보컬 톤, 달라졌어. 어제 합동 연습 때 들었거든. 호흡이 좋아졌더라."

민지는 가슴이 철렁했다. '벌써... 눈치챘어?'

"그냥... 운 좋았나 봐..."

"운? ㅋ" ARIN이 비웃었다.

"나중에 알려줘. 뭐 하는지. I'm curious."

![연습실 - ARIN이 멀리서 MIRA 관찰](placeholder)
*오전 10시. 연습실 A. MIRA가 혼자 보컬 연습 중. 거울 너머로 ARIN이 에어팟 빼고 집중해서 듣고 있다.*

오전 10시. 연습실 A.

민지는 혼자 보컬 연습을 했다. ChatGPT가 알려준 발성법을 연습했다.

"도~ 레~ 미~ 파~ 솔~"

솔 음이 안정적으로 나왔다. 어제보다 좋았다.

"오... ㄹㅇ 좋아졌네?"

민지는 메모 노트를 펼쳤다. 체크박스에 표시했다.

그때 문이 살짝 열렸다. ARIN이 거기 있었다. 에어팟을 빼고 있었다.

"...!" 민지는 노트를 덮었다. 하지만 늦었다.

ARIN은 연습실로 들어왔다. 민지 앞에 섰다.

"지금... 뭐 한 거야?"

"그냥... 연습..."

"아니. 그게 아니라." ARIN이 민지보다 5cm 큰 키로 내려다봤다.

"네 보컬. 진짜 달라졌어. 2주 전이랑 완전 다른데?"

"...운이 좋아서..."

"거짓말. 운으로 안 돼. 나도 6개월 걸렸거든, 그 정도 늘려면."

ARIN의 눈이 민지 손에 든 노트로 향했다.

"그거. 뭔데?"

민지는 노트를 가슴에 안았다.

"개인 거라... 보여줄 수 없어..."

"흠." ARIN은 민지를 뚫어지게 봤다. 그리고 돌아섰다.

"Okay. 안 알려주면 말고. 근데 말이야..."

문 앞에서 ARIN이 멈췄다.

"네가 뭘 하든, 나도 할 수 있어. 너보다 더 잘할 수 있고."

문이 닫혔다. 민지는 가슴이 두근거렸다.

'ARIN 언니... 질투...?'

"혹시... 뭔가 새로운 걸 시작한 거야?"

"응... 고마워 언니."

CHAEWON이 밥을 먹으려는데, 손톱이 보였다. 물어뜯은 흔적. 민지는 그걸 봤다.

'언니도... 힘든 거네...'

## 💡 Hands-On Tutorial: 협업 대화 시뮬레이션

**Real-world situation**: MIRA는 ChatGPT로 실력이 늘고 있지만 ARIN에게 들켰다. "뭐 하는 거냐"고 계속 묻는 상황. 알려줘야 할지 고민이다. ChatGPT에게 "대화 시뮬레이션"을 요청해 상대를 설득하는 연습을 할 수 있다.

**Copy-paste prompt**:
```
너는 {상황, 예: "연습생"} 커뮤니케이션 전문가야.

**나(A):** {이름, 예: "민지"}, {성격, 예: "불안형"}, 비밀: {내용, 예: "AI 훈련 중"}
**상대(B):** {이름, 예: "서연"}, {성격, 예: "직설적"}, 질문: {내용, 예: "너 뭐 해?"}

B의 질문 3단계와 A의 답변 옵션 3가지(솔직/모호/회피), 각 결과 예측, 추천 전략을 대화 시뮬레이션 형식으로 작성해줘.
```

**What you'll get**: 상대방 질문 3단계와 답변 옵션별 결과 예측, 추천 전략이 나온다.

**Try it yourself**:
- [ ] https://chat.openai.com 접속
- [ ] 프롬프트 복사-붙여넣기, {placeholders} 수정
- [ ] Enter로 실행, 시뮬레이션 결과 읽기
- [ ] 추천 전략 선택, 실제 상황에서 사용
- [ ] 결과를 ChatGPT에 "이렇게 말했더니 {반응}" 입력해 피드백
- [ ] 필요시 "더 구체적인 대사 예시 줘" 추가 요청

**Example result**:
```
## 대화 시뮬레이션

### Question 1 (간접)
**서연**: "너 뭔가 다른데? 비법 있어?"
답변 A) 솔직: "AI 쓰고 있어" → 서연 관심 80%
답변 B) 모호: "연습 방법 바꿨어" → 서연 계속 물음 90%
답변 C) 회피: "운이 좋아서" → 서연 의심 100%

### Question 2 (직접)
**서연**: "그 노트 보여줘."
답변 A) 솔직: "AI 계획표야" → 협업 가능성 60%
답변 B) 모호: "훈련 기록이야" → 긴장 지속
답변 C) 회피: "안 돼" → 갈등 심화

### Question 3 (압박)
**서연**: "알려줄 거야 말 거야?"
답변 A) 솔직+제안: "알려줄게. 같이할래?" → 수락 70%
답변 B) 미룸: "나중에..." → 서연 스스로 찾음 80%
답변 C) 거절: "내 방식이야" → 관계 악화

🎯 추천: Question 2에서 Option A (부분 공개 + 협업 제안)
```

## 🎯 Learning Concept: "혼자 하면 외로운 기 같아"

**Level 1 Concept**: 협업의 시작

MIRA는 처음엔 ChatGPT 계획을 혼자 지키려 했다. 하지만 ARIN의 질문과 자신의 외로움을 마주하며 깨달았다. 혼자 하는 성장은 한계가 있다는 것을.

AI 도구는 개인에게 강력하지만, 사람과 함께 쓸 때 더 강력하다. MIRA가 ChatGPT로 ARIN을 설득하는 법을 배운 것처럼, AI는 "함께 잘하기"를 돕는다.

**Why it matters in 연습생 생활**: 연습생은 경쟁자이면서 동료다. AI 도구를 혼자 쓰면 개인 성장. 같이 쓰면 팀 성장. MIRA와 ARIN이 AI로 협업하면, 둘 다 더 빨리 늘고, 팀워크도 좋아진다.

**Remember (MIRA's words)**: "같이 하자. 혼자 하면 외로운 거 같아서..."
