## 🎨 파일 구조

각 에피소드는 **2개 파일**로 구성:

### 1. Story 파일 (`episode_N_story.md`)
```markdown
# Episode N: 제목

스토리 본문 (3,500-5,000자)
- Opening
- Development
- Tutorial (통합)
- Learning Concept
- Resolution

![장면](placeholder) ← 일러스트 배치 위치
```

### 2. Illustrations 파일 (`episode_N_illustrations.md`)
```markdown
# Episode N Illustrations

## 🎨 Scene 1: [제목]
NovelAI Prompt:
[캐릭터, 포즈, 배경, 무드 등 상세 프롬프트]
--ar 16:9 or 2:3

[Scene 2-8 동일 구조]

## 📊 Illustration Analysis
- Priority Scoring (1-10점)
- Density Justification
- SIGNATURE PATTERNS 체크
```

## 📖 스토리 개요

### 시놉시스
**JYP 연습생 민지, 73점으로 탈락 위기.**  
→ Claude AI 발견, 체계적 시스템 구축  
→ 팀원들과 함께 성장 (갈등과 극복)  
→ 3개월 만에 83점 달성, 데뷔 확정

### 핵심 메시지
1. **AI는 도구**: 시스템을 만들어주지만 실행은 사람
2. **꾸준함**: 매일 0.1점씩 100일 = 10점 상승
3. **팀워크**: 혼자는 안 되지만 함께는 가능
4. **투명성**: 데이터 기반 의사결정 (숫자로 말한다)
5. **회복**: 번아웃 방지가 지속 가능성의 핵심

### 주요 시스템
1. **팀 루틴 (Episode 4)**: 6시 기상, 일일 계획, 컨디션 체크
2. **개인 플랜 (Episode 6)**: 5명 맞춤형 훈련
3. **통합 스케줄 (Episode 11-14)**: 개인+팀 균형
4. **주간 회고 (Episode 15)**: 매주 성찰
5. **유지 전략 (Episode 16)**: 80점 지키기

## 💡 활용 가이드

### 1. 웹툰/웹소설 제작자
- **Story 파일**: 그대로 각색 또는 참고
- **Illustrations**: NovelAI로 일러스트 생성
- **구조**: 3막 구조, 감정 아크, 클리프행어 참고

### 2. AI 교육자
- **Tutorial**: 업무 AI 활용 사례
- **Pattern**: A/B/C/D 4가지 Tutorial 형식
- **실전**: 실제 적용 가능한 프롬프트

### 3. 작가 지망생
- **SIGNATURE PATTERNS**: 흥행 요소 5가지
- **Character Development**: 5명의 성장 아크
- **Dialogue**: 캐릭터별 말투 차별화

### 4. 기업 교육 담당자
- **시스템**: 개인→팀→조직 확산 과정
- **측정**: 데이터 기반 성과 관리
- **지속**: 번아웃 방지, 장기 성장
